(Google Bulk Image Downloader)


ensure the file locations are correct and the index file is in a folder named templates which lies in the directory of the main code

Steps:
1: replace senders email
2: create an app password and use it while also enabling 2FA
3: run the downloader app.
4: open the IP which appears.
5: Enter your search term and wait for the results!


EXAMPLES are in the RAR files, for the downloaded images, processed images, and the zipped image files for 4 search term exapmles.

Please note that the Drive API has not been set yet, so a large number of images will be downloaded and BG removed, and it will be zipped. However, since email only allows 25MB of attachments, a drive API should be used, which has not yet been integrated
